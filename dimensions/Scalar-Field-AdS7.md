# Scalar Field in AdS7: Bulk-to-Boundary Propagation

A **massive scalar field** in AdS7 illustrates the holographic dictionary — bulk field φ(z,x) ↔ boundary operator O(x) with dimension Δ.

#### Klein-Gordon Equation in AdS7 (Poincaré Coordinates)
(□ - m²) φ = 0

In Poincaré metric ds² = R²/z² (η_μν dx^μ dx^ν + dz²)

The equation becomes:
z² ∂_z (z⁻⁵ ∂_z φ) + z² □_6 φ - m² R² φ = 0

#### Near-Boundary Behavior (z→0)
φ(z,x) ~ z^{Δ_-} A(x) + z^{Δ_+} B(x)

where Δ± = 3 ± √(9 + m² R²)

- Δ+ = conformal dimension of operator O
- A(x) = source (non-normalizable mode)
- B(x) = vev ⟨O(x)⟩ (normalizable mode)

#### Bulk-to-Boundary Propagator
G(z,x; x') ~ (z / (z² + (x-x')²))^{Δ}

#### Holographic Renormalization
Counterterms on boundary remove divergences — vev B(x) finite.

#### Example: Free Scalar (m=0)
Δ = 6 or 3 (quantization choices)
