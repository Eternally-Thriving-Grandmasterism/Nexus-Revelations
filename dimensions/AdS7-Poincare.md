# Poincaré Coordinates for AdS7: Boundary Perspective

**Poincaré coordinates** provide a useful slicing of AdS7 that makes the conformal boundary explicit — ideal for holographic calculations.

#### Poincaré Metric for AdS7
ds² = \frac{R²}{z²} ( -dt² + dx_i dx^i + dz² )   (i=1 to 5)

where:
- z > 0 (radial coordinate, z=0 is conformal boundary)
- x_i: 5 spatial directions on the boundary
- t: time

#### Key Features
- Conformal boundary at z→0: metric → flat 6D Minkowski
- Horizon at z→∞: proper distance infinite
- Isometries: Poincaré group of 6D Minkowski + scale transformations

#### Relation to Global Coordinates
- Poincaré patch covers half of global AdS
- Useful for plane-wave states and boundary CFT correlation functions

#### Holographic Dictionary
- z = energy scale in boundary CFT
- Bulk fields φ(z,x) ↔ boundary operators O(x) with dimension set by mass
